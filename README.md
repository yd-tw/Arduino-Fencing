## 專案介紹
這個專案是在生活科技課程創建的。這個專案作為課程期中評分，以及Arduino課程的實際應用。這個專案於期末獲得老師給予98分評價。

## 技術細節

- Nrf24l01無線通訊模組
- OLED液晶顯示器
